== Description ==

Languages: English.

== Installation ==

1. Upload `wp-permalinks-export` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In 'Plugins', click on 'Export Permalinks'.
